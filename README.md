GroupDNA: WhatsApp Chat Analyzer
GroupDNA is a lightweight Python-based log parser and analytics tool designed to process exported WhatsApp chat data (hostel_bois.txt). It transforms raw message logs into deeply insightful behavioral analytics, mapping group communication patterns, response tendencies, and distinct user archetypes over time.

🚀 Features
Robust Preprocessing: Cleans and formats raw chat data into a structured Pandas DataFrame, handling timestamps, system notifications, and varying chat states.

Media & Activity Tracing: Counts and tracks media attachments (<Media omitted>) and monitors user-specific message deletion habits.

Response Analytics: Computes group responsiveness statistics, highlighting your fastest repliers, slowest repliers, and longest silent streaks.

Behavioral Archetypes: Identifies user communication patterns to dynamically classify members into archetypes.

Visual Reports: Generates an ASCII-based terminal dashboard displaying messaging trends and text analytics cleanly.
